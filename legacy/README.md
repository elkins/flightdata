# Legacy Code

This directory contains deprecated Python 2.7 code from the original FlightRadar24 implementation.

**These files are kept for historical reference only and are not maintained.**

## Files

- `flightradar24.py` - Original FlightRadar24 API client (Python 2.7)
- `logger.py` - Original logging utilities (Python 2.7)
